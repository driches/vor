<section class="ccr-section">
  <h2 class="ccr-h2">60-second setup</h2>
  <p class="ccr-sub">Drop one workflow file in your repo, add your API key as a secret, and trigger a review by hand.</p>

```yaml title=".github/workflows/vor.yml"
name: Vor
on:
  workflow_dispatch:
    inputs:
      pr_number:
        description: 'PR number to review'
        required: true

permissions:
  contents: read
  pull-requests: write

jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with: { fetch-depth: 0 }
      - uses: driches/vor@v0
        with:
          anthropic_api_key: ${{ secrets.ANTHROPIC_API_KEY }}
          pr_number: ${{ inputs.pr_number }}
```

  <p class="ccr-step">Prefer OpenAI? Swap in <code>openai_api_key</code> and set <code>model: gpt-4.1</code> — the provider is inferred from the model id. Then: <strong>Actions → Vor → Run workflow → enter the PR number</strong>. A sticky review appears within minutes.</p>
